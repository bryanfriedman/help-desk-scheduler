2.3. Employee Page

This is the Employee page:

The buttons on the menu on the left allow you to do the following things:

  • The "Preferences" button allows you to Set Schedule Preferences.
  • The "Password" button allows you to Change Your Password.
  • The "Shift Cover" button allows you to Request or Respond to Shift Cover.
  • The "View/Print" button allows you to View/Print a Schedule.
