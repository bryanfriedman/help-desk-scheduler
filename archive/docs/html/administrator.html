2.2. Administrator Page

This is the Administrator page:

The buttons on the menu on the left allow you to do the following things:

  • The "Accounts" button allows you to Administer Accounts.
  • The "Priorities" button allows you to Set Priorities.
  • The "New Schedule" button allows you to Create a New Schedule.
  • The "Generate Schedule" button allows you to Generate a Schedule.
  • The "Edit Schedule" button allows you to Manually Adjust a Schedule.
  • The "View/Print" button allows you to View/Print a Schedule.
